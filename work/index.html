My work.

Zircus Apparel

I designed and implemented a full-stack solution for Zircus Apparel. On the front-end, I used Jekyll to produce a static site for strong SEO and ease of Internationalization. In addition, I used Web Components to provided a progressive, enhanced user experience.

The back-end API is written in Node.js, using the Express framework. It integrates with both PayPal and Stripe's APIs to offer multiple payment solutions, along with integrating with Canada Post's older XML-based REST API for real-time shipping calculations and automated label-making.

The back-end also includes a React-powered admin panel, written to enable Zircus Apparel to track their inventory and manage orders.

I have experience with various JavaScript technologies, such as React, Vue.js, Node.js and Native Web Components. I am also confident with CSS and HTML.
